# Agri-Yield Exploratory Data Analysis

This repository contains the Exploratory Data Analysis (EDA) work done as part of my internship project on agricultural yield data.

# Tasks Involves
- FAOSTAT dataset cleaning
- Crop dataset cleaning
- Merging multiple datasets
- Duplicate value checking
- Missing value handling
- Boxplot analysis
- Outlier removal using IQR method
- Correlation heatmap
- Pairplot analysis
- Export of final cleaned dataset

# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
